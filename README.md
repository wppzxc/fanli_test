# fanli_test

## download mingw 
```
https://codeload.github.com/go-vgo/Mingw/zip/master
```

## set env
```
Path = D:\<your mingw parent path>\mingw\bin
```
## build
```
go build -o fanli.exe
```

## run
```
fanli.exe -u wxOpenId -t "user1,user2" -d 10
# wxOpenId like os_Ph0j7eHfGJhowF8E-_kJc1fiM
```
