# 使用教程
## 安装依赖
```
下载依赖，地址：
https://codeload.github.com/go-vgo/Mingw/zip/master

将压缩包解压到本地

将 third_party 路径下的 file2clip.exe 文件放到上一步解压后的 bin 路径下

将解压后的 bin 路径（如：D:\software\sdk\mingw\bin）加入到系统环境变量 Path中

打开 cmd 窗口，输入 gcc -v 敲回车，如果有大量返回则依赖安装成功
```
